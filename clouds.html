Clouds, Services and Servers

What does it do?

Clouds, Services and Servers are used within the corporate world to provide a user base with the tools that they require to get their work completed. If you think of most environments on the interest, websites, collaboration sites and anything else it would be using some form of cloud or physical server environment.

With the introduction of cloud computing the cost of running a business can be reduced and small startups now have the opportunity to start with less overheads.

What is the state of the art of this new technology?

Microsoft Azure

Microsoft Azure is leading the cloud server enterprise. The easy to use interface, cost analysis and flexibility of this service makes it the major provider for a multi-domain level environment to a single standalone environment.

Amazon Web Services

Amazon over the years have increased their presence in the cloud market with products such as AMS (Marketing Services) and AWS (Web Services).

What can be done now?

Setting up office environments (IaaS)

With the advancements in cloud technology, Internet capability and faster processing power from hardware an office that would normally run on physical servers on premise can now be 100% hosted in a cloud environment.

What is likely to be able to do be done soon (say in the next 3 years)?

We will see SaaS (Software as a service) being developed closely alongside of AI/Machine learning. This will be able to create a service that can be flexible, reliable, resilient and future proofed.

What technological or other developments make this possible?

The development of faster computing power, access to faster and more stable internet has seen the rise of cloud services and servers. The ease of use and access by users has concreted this into a part of everyday life.

Nearly every type of IT solution we are using today has some sort of cloud environment or access.

What is the likely impact?

With the increase of easy to access cloud-based services we will see more and more user-friendly environments that will be able to be accessed more often. Such as Google Docs, and MS Teams these services have already made a massive impact on how we all work together.

What is the potential impact of this development?

The overall decline in physical infrastructure for office environments. The move towards cloud environments will bring the need to only have a way of accessing your enterprise such as BYOD or virtual sessions from a very basic computer.

What is likely to change?

How we think about hosting services, setting up a new work environment, playing a video game or even hosting a meeting. All these things will be moved in one way or another to a cloud platform. The initial start is there and with the current climate of the world, moving towards a remote platform will be key for certain businesses to survive.

Which people will be most affected and how?

The increase in cloud services will be a technology that will affect everyone in every way of life. It is already utilized for so many things that most people don't even realize they use it for.

Will this create, replace or make redundant any current jobs or technologies?

Within the administration of these environments we will see a reduction in onsite personal being replaced with remote offices and administrators who can fix incidents remotely.

There will be a reduction in a workforce if AI and Machine learning starts to get incorporated into working environments.
